    Начать тест
    Задание №1
    Как объявить переменную с использованием ключевых слов var, let и const?
    let x = 10
    variable y = "Hello";
    const z;
    Ответить
    Задание №1
    Правильно!
    Переменная в JavaScript - это именованное место для хранения данных. Она объявляется с помощью ключевых слов var, let и const: var (устарело) - имеет функциональную область видимости и подвержено всплыванию переменных (hoisting). let (ES6) - имеет блочную область видимости, значение переменной можно изменить. const (ES6) - обозначает константу, значение которой нельзя изменить после присвоения.

    Как объявить функцию и как вызвать её?
    let multiply = (a, b) => { return a * b; }, multiply(2, 4);
    function add(a, b) { return a + b; }, let result = add(3,5);
    function greet(name) { console.log(name); }, greet();
    Ответить
    Задание №2
    Функция в JavaScript - это блок кода, который может быть вызван для выполнения определенной задачи или вычисления значения. Объявляется с помощью ключевого слова function. Вызывается с указанием имени функции и передачей аргументов.

    Как использовать операторы if, else if и else для выполнения различных действий в зависимости от условий?
    const num = 10; else if(num > 0) {...} if(num < 0) {...} else {...}
    const num = 10; else (num > 0) {...} else if (num < 0) {...} if{...}
    const num = 10; if (num > 0) {...} else if (num < 0) {...} else {...}
    Ответить
    Задание №4
    Условные операторы в JavaScript используются для выполнения различных действий в зависимости от условий. Операторы if, else if и else позволяют выбирать между разными вариантами выполнения кода в зависимости от значений условий.

    Задание №5
    Как использовать циклы for и while для выполнения повторяющихся действий?
    let i = 0; while (i <= 10) { console.log("Число: " + i); }
    for (let i = 0; i < 5; i++) { console.log("Повторение №" + (i + 1)); }
    for (let i = 10; i > 0; i--) { console.log("Обратный отсчет: " + i); }
    Ответить
    Задание №5
    Правильно!
    Циклы в JavaScript используются для выполнения повторяющихся действий. Цикл for используется для выполнения действий определенное количество раз, пока выполняется условие. Цикл while выполняется до тех пор, пока условие истинно.
